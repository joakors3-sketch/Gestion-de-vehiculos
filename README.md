Sistema de Gestión de Vehículos
Descripción
Aplicación de consola desarrollada en C# utilizando Programación Orientada a Objetos (POO).
El sistema permite registrar distintos tipos de vehículos, mostrar su información y simular acciones de aceleración y frenado. Cada tipo de vehículo posee un comportamiento diferente al modificar su velocidad.
Funcionalidades
Registrar vehículos.
Mostrar información de los vehículos.
Simular aceleración.
Simular frenado.
Mostrar cambios de velocidad en tiempo real.
Gestionar distintos tipos de vehículos mediante polimorfismo.
